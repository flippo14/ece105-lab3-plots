# ece105-lab3-plots

A Python project that generates synthetic temperature sensor data and produces publication-quality visualizations using NumPy and matplotlib.

## Usage
This will generate `sensor_analysis.png` in the current directory containing three plots: a scatter plot, histogram, and box plot.

## Example Output

The script produces three side-by-side plots:
- **Scatter plot**: Temperature readings from both sensors over time
- **Histogram**: Overlaid distributions of both sensors with mean lines
- **Box plot**: Side-by-side comparison of sensor distributions
